# LaTeX公式符号

## 1. 字母上下标

- $\overline{a}$ :    	\overline{a}
- $\hat{a}$ : 	   \hat{a}
- $\widehat{a}$ :		\widehat{a}
- $\widetilde{a}$ :		\widetilde{a}
- $\dot{a}$ :		\dot{a}
- $\ddot{a}$ :		\ddot{a}
- $a_{1}^{2}$ :      a_{1}^{2}
- $\sum_{i=1}^{n}$ :      \sum_{i=1}^{n}
- $\sum\limits_{i=1}^{n}$:            \sum\limits_{i=1}^{n}
- $\mathop{\Pi}\limits_{i=1}^{n}$ :           \mathop{\Pi}\limits_{i=1}^{n}
